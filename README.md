# t# 📈 TrendApp — Real-Time Brand Sentiment Engine

An interactive social listening and sentiment analytics platform that allows businesses to track consumer mentions, monitor sentiment polarity, and analyze social volume trends in real-time.

---

## 🌟 Key Features

* **🏷️ Custom Brand Onboarding:** Personalized onboarding flow allowing users to input any brand name and domain to scope analytics.
* **📊 Interactive Chart Visualizations:** Dynamic line charts (volume trends) and doughnut charts (polarity split) powered by Chart.js.
* **💬 Real-Time Mention Stream:** Simulated telemetry feed featuring positive, neutral, and negative sentiment tagging.
* **🔍 Search & Polarity Filters:** Instant client-side text searching and category filtering.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Light Mode / Glassmorphism), JavaScript (ES6+)
* **Data Visualization:** Chart.js
* **Backend Integration:** Python (Flask API service in `app.py`)
* **Hosting:** GitHub Pages
